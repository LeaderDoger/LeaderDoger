# 容斥原理

通过二进制枚举所有可能出现的情况，再通过1的数量判断符号

```c++
//容斥原理
    int res=0;
    for(int i=0;i<1<<n;i++){
        //多放一个小球防止隔板法的隔板在一起
        //所以有m+n-1个空，n-1个板
        LL a=m+n-1,b=n-1;
        int sign=1;
        for(int j=0;j<n;j++)
            if(i>>j&1){
                //确定符号
                sign*=-1;
                //计算计算方案要用的数据
                a-=A[j]+1;
            }
        //概率或方案数相加
        res=(res+C(a,b)*sign)%mod;
    }
```
